# 🐍 Python Basics Tutorial

A collection of beginner-friendly Python programs covering core programming concepts — perfect for lab practice, assignments, and interview prep.

---

## 📁 Programs Included

| File | Description |
|------|-------------|
| `fibonacci.py` | Print Fibonacci series up to N terms |
| `factorial.py` | Calculate factorial using recursion |
| `prime.py` | Check if a number is prime |
| `armstrong.py` | Check for Armstrong numbers |
| `palindrome.py` | Check if a word/number is a palindrome |
| `caesar_cipher.py` | Encrypt and decrypt text with Caesar cipher |
| `binary_decimal.py` | Convert between binary and decimal |
| `calculator.py` | Simple arithmetic calculator |
| `guessing_game.py` | Number guessing game with random module |
| `string_operations.py` | Common string manipulations |

---

## 🚀 How to Run

Make sure Python 3 is installed, then run any file:

```bash
python fibonacci.py
```

No external libraries needed — all programs use Python's standard library only.
